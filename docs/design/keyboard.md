# Keyboard Shortcuts

Planned keyboard interaction map for Helling WebUI.

## Global

- `Cmd+K` / `Ctrl+K`: open command/search palette
- `/`: focus current page filter/search input
- `g` then key: go-to navigation shortcuts (reserved for future rollout)

## Page-Level Principles

- Shortcut conflicts with browser/system bindings must be avoided.
- Shortcuts must be discoverable via UI help.
- Keyboard interactions must remain optional and non-blocking.
