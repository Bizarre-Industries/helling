# ADR-045: APT Repository Tooling

> Status: Proposed (2026-04-20)

## Context

APT publication tooling is marked TBD in infrastructure standards.

## Decision

Pending decision between:

- aptly-based managed repository workflow
- static-file APT repository publication via GitHub Pages

## Decision Criteria

- Release signing and trust model
- Operational complexity
- Automation reliability in CI
- Long-term maintenance overhead

## Next Step

Run a short comparison spike and accept one approach.
