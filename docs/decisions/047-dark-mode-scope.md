# ADR-047: Dark Mode Scope for v0.1

> Status: Proposed (2026-04-20)

## Context

Dark mode behavior for v0.1 has been implied but not explicitly decided.

## Decision

Pending explicit scope decision:

- v0.1 light-only
- v0.1 includes both light and dark themes

## Decision Criteria

- Delivery scope and QA overhead
- Accessibility contrast validation
- Design-system readiness

## Next Step

Choose scope and document acceptance criteria.
