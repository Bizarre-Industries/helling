# ADR-048: Mobile Scope for v0.1

> Status: Proposed (2026-04-20)

## Context

Mobile support expectations are currently ambiguous.

## Decision

Pending explicit scope decision:

- v0.1 desktop-only
- v0.1 responsive/mobile-supported

## Decision Criteria

- Product expectations and user workflow assumptions
- UI complexity and testing cost
- Accessibility and interaction parity

## Next Step

Define target form factors and required acceptance tests.
